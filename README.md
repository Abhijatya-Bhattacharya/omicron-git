# Omicron Research Assistant - G34 MSIT

A powerful research assistant built with Dart and Flutter that leverages local LLM capabilities through Ollama CLI, providing comprehensive research capabilities while maintaining user privacy and anonymity.

## üåü Features

- **Local LLM Integration**: Harnesses the power of Ollama CLI for on-device language model processing
- **Privacy-First Architecture**: Anonymous research capabilities using the proprietary Proanzr architecture
- **Hybrid Intelligence**: Combines proprietary and open-source solutions for optimal performance
- **One-Click SLR Generation**: Automated Systematic Literature Review (SLR) PDF creation with a single click
- **Up-to-Date Information**: Real-time access to current research and information while maintaining anonymity
- **Cross-Platform**: Built with Flutter for seamless operation across desktop and mobile platforms

## üèóÔ∏è Architecture

Omicron utilizes a unique hybrid architecture:

- **Frontend**: Flutter SDK for responsive, cross-platform UI
- **Backend Logic**: Dart language for robust application logic
- **LLM Engine**: Ollama CLI for local language model inference
- **Research Layer**: Proanzr architecture (proprietary + open-source) for anonymous, up-to-date information retrieval

### Generating an SLR

1. Launch Omicron
2. Enter your research topic or query
3. Provide More instructions
4. Click "Generate SLR"
5. Wait for the automated research and compilation process
6. Download your comprehensive SLR PDF
